# Ambern Ltd — Official Website

**Ambern Ltd** is an international digital marketing agency headquartered in London, England & Wales (Company No. 16622550), specialising in performance-based affiliate marketing across Tier-1 global markets.

## Tech Stack

- **Framework:** React 19 + TypeScript
- **Styling:** Tailwind CSS 4 + custom CSS
- **Routing:** Wouter
- **Build Tool:** Vite 7
- **Deployment:** Vercel (recommended)

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — Hero, Services, Networks, Stats, FAQ, Contact |
| `/company` | Company Information — Official Companies House data, Directors, Timeline |

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Deployment on Vercel

1. Connect this repository to Vercel
2. Set **Framework Preset** to `Vite`
3. Set **Build Command** to `pnpm build`
4. Set **Output Directory** to `dist`
5. Deploy
